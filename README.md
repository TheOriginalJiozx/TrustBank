TrustBank

En simpel bank management webapp.


Krav

Node.js 18+ (inkluderer npm)

Tjek versioner:

node -v
npm -v

Installation
git clone <https://github.com/TheOriginalJiozx/TrustBank.git> Trustbank
cd trustbank

npm install

Start (Development)
npm start
# Kører på http://localhost:3000

Build (produktion)
npm run build


Hvis port 3000 (Frontend) allerede er i brug: PORT=3002 npm start
Hvis port 3001 (Backend) allerede er i brug: PORT=3003 npm run dev
